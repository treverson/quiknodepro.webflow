 Instances

Creating An Instance

There are two ways of creating new QuikNode instances:
‍
1. From the Dashboard. Simply click 'New Instance' button under Quik Create.


new instance

2. From Instances section. Click on the '+' button under Instances.



You will be presented with a configuration modal. Instance name will be generated automatically. You may enter an optional instance description and click 'Launch'.
Once your instance is spun up, it will be listed in Instances section.
‍

Accessing your instance. Web Socket and HTTP access. Validators

In order to connect to your Instance, use the generated Web Socket and/or HTTP access links and credentials. To find access URLs, simply click on your instance.
Click on any of the links under DEVELOPER ACCESS and they will be automatically copied to your clipboard.

The VALIDATORS menu allows you to add additional security validation for accessing your instance. Select the type of validation you wish to use from the dropdown, enter your referer URL and/or http token and click Save.
To remove a Validator - click on the circular trash icon next to the input field.


Instance operations: Suspend, Remove, Clone.

Instances can be temporarily suspended, removed permanently or cloned. Cloning an instance will create a new instance with same VALIDATORS and DESCRIPTION as current instance. These operations are featured under the Instance name.


Managing multiple instances.

Multiple instances can be easily managed from the Instances section.
You can change your view by clicking the Grid button.

Instances can be sorted by Date of Creation and Name. Clicking on the arrow next to the dropdown specifies ascending or descending order.

While in list view Bulk Actions can be performed on multiple instances by selecting them and choosing an appropriate action from the dropdown.

Monitoring instance activity (Metrics).

Instance activity charts can be accessed either from the Dashboard or by viewing/clicking on the mini-chart for each Instance.

To customize metrics view and view particular metrics use the dropdown filters.

Note that charts can be exported as images or PDF files by clicking on the context menu on top right corner.

 Users

Adding a new User

You can authorize multiple users to use your QuikNode Pro account. Users can have limited permissions or full access to your account and manage other users. There are two ways to add a new User:

1. From the Dashboard, under Quik Create, simply click the Add New User button.


new user


2. You can add a new User from Users section by clicking this button:
‍




Specify user's email, choose a password and hit Save.
To edit Users go to the Users section in main menu and click on the user box or row.
‍

User Permissions

Your account users can have limited access permissions or full Admin rights to manage other users. Check the list of available options under PERMISSIONS heading while editing the user.

User Operations: Suspend, Remove, Clone

Users can be temporarily suspended, deleted permanently or cloned. Cloned users will inherit the same permissions and API keys from the original.

Managing multiple users

Multiple Users can be easily managed from the Users section.
You can change your view by clicking the Grid button.

Users can be sorted by Date of Creation and Name. Clicking on the arrow next to the dropdown specifies ascending or descending order.

While in list view Bulk Actions can be performed on multiple Users by selecting them and choosing an appropriate action from the dropdown.

 Security

Managing aPI keys

API keys are used to provide programmatic access to the QuikNode Pro system. To create a new API key, access the Security section, select API keys tab, provide your own Secret key and click Generate.

To revoke an existing API key, simply click on the circular Trash icon next to the key.

One Time Passwords (OTP)

One-Time-Passwords (OTP) are single-use access tokens generated by the system in the form of a QR code. These can be used to gain access to your QuikNode Pro account without compromising your main credentials. To generate an OTP, select ONE TIME PASSWORDS tab, enter description and hit Generate.
The created QR code contains your OTP, it will only be shown once.

To revoke an OTP, simply click the circular trash icon next to it in the list.
